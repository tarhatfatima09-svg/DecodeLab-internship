# DecodeLab Internship Projects

This repository contains the projects completed during my internship at DecodeLab, focusing on data analytics, business insights, and dashboard development using Microsoft Excel.

## Project 1: Sales Analysis Using Pivot Tables

### Overview

Performed sales data analysis using Pivot Tables to identify product performance, revenue trends, customer purchasing patterns, and business growth indicators.

### Analysis Performed

* Sales by Product and Order Status
* Revenue by Channel and Payment Method
* Orders and Average Order Value by Year
* Monthly Revenue Trend Analysis

### Tools Used

* Microsoft Excel
* Pivot Tables
* Charts and Graphs

### Key Skills Gained

* Data Analysis
* Data Summarization
* Business Reporting
* Excel Functions and Pivot Tables
